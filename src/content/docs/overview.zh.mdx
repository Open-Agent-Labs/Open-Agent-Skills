# 概览

一个简单、开放的格式，为 AI Agent 提供新能力和专业知识。

Open Agent Skills 是包含指令、脚本和资源的文件夹，Agent 可以发现并使用它们来更准确、更高效地完成任务。

## 为什么需要 Open Agent Skills？

Agent 的能力越来越强，但往往缺乏可靠完成实际工作所需的上下文。Skills 通过让 Agent 按需加载程序性知识以及公司、团队和用户特定的上下文来解决这个问题。拥有一组 skills 的 Agent 可以根据正在处理的任务来扩展其能力。

**对于 skill 作者**：一次构建能力，部署到多个 Agent 产品。

**对于兼容的 Agent**：支持 skills 让终端用户可以开箱即用地为 Agent 添加新能力。

**对于团队和企业**：将组织知识封装在可移植、版本控制的包中。

## Open Agent Skills 能实现什么？

- **领域专业知识**：将专业知识打包成可重用的指令，从法律审查流程到数据分析管道。
- **新能力**：为 Agent 提供新能力（例如创建演示文稿、构建 MCP 服务器、分析数据集）。
- **可重复的工作流**：将多步骤任务转变为一致且可审计的工作流。
- **互操作性**：在不同的 skills 兼容 Agent 产品中重用相同的 skill。

## 采用情况

Open Agent Skills 已被领先的 AI 开发工具所支持。

## 开放开发

Agent Skills 格式最初由 [Anthropic](https://www.anthropic.com/) 开发，作为开放标准发布，并已被越来越多的 Agent 产品采用。该标准对更广泛的生态系统开放贡献。

[在 GitHub 上查看](https://github.com/agentskills/agentskills)
