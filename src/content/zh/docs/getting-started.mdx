# 快速开始

欢迎使用 **Open Agent Skills**！本指南将帮助您快速上手。

## 前置要求

在开始之前，请确保已安装以下工具：

- Node.js 18.17 或更高版本
- pnpm（推荐）或 npm

## 安装

1. 克隆仓库：

```bash
git clone https://github.com/your-repo/open-agent-skills.git
cd open-agent-skills
```

2. 安装依赖：

```bash
pnpm install
```

3. 启动开发服务器：

```bash
pnpm dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看效果。

## 项目结构

```
src/
├── app/[locale]/     # 支持国际化的 App Router
├── components/       # 可复用组件
├── content/          # MDX 内容文件
├── i18n/             # 国际化配置
└── lib/              # 工具函数
```
